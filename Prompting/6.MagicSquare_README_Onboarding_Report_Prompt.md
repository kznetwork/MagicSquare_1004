## 1순위: 구현·검증 작업의 단위로 쓰기 — PRD

**`docs/5.PRD_MagicSquare_4x4_TDD.md`**(또는 베이스라인 보고서에 적힌 `docs/PRD_MagicSquare_4x4_TDD.md`와 동일 계열)가 가장 적합합니다.
